# api_testtool

PowerShell script to test APIs from Postman collections, supporting single calls, project-wide tests, and stress testing.

## Features
- Single API test
- Run all APIs in a collection
- Stress testing mode (100 requests)
- Logs requests and response metrics

## Installation
```powershell
git clone https://github.com/JohannesWaldeck/api_testtool.git
cd api_testtool
```
## Usage
Run the script in PowerShell:
```powershell
.\api_testtool.ps1
```
